# Прогресс разработки (SEMD Engine)

## Реализовано
- [x] **Базовая структура проекта**: Pydantic-модели, Jinja2-шаблоны, Data Fetcher.
- [x] **Генерация CDA**: Поддержка шаблонов v290_r7 (Протокол консультации).
- [x] **XSD Валидация**: Интегрирована в `CDARenderer` через `lxml`.
- [x] **Schematron Валидация**: 
    - Решена проблема поддержки XSLT 2.0 (стандартные Python-библиотеки поддерживают только 1.0).
    - Интегрирован `schxslt-cli.jar` для полноценной проверки бизнес-логики СЭМД.
    - Реализован обход проблем с неймспейсами (удаление `xmlns` перед проверкой).
- [x] **Тестирование**: Созданы интеграционные тесты для проверки структуры и логики документов.
